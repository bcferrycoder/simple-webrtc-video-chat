Stackato / WebRTC Demo

This is a basic video/audio chat demo using WebRTC, originally forked from https://github.com/keithnorm/simple-webrtc-video-chat.git.

To create a chat room simply choose a name for it and append it to the URL string you used to access this page. Or you can join the Stackato chat room. Then provide the full URL to another user. When the second user visits this same URL, and grants access to the camera/microphone, two video windows will appear, one for each participant. Use headphones or mute your mic, otherwise you'll be accosted with feedback.

Limitations

  • Audio feedback: mute your mike
  • Tested only on Chrome
  • Scaling out multiple instances of this app will cause problems as each app instance maintains its own chat rooms. Works fine with one app instance.
